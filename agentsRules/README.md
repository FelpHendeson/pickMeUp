# Agents Rules

Esta pasta concentra contexto e regras operacionais para agentes ou pessoas que forem evoluir o projeto.

Arquivos:
- `contexto.md`: visao geral do produto, estrutura e fluxo principal.
- `pre-analise.md`: checklist antes de alterar o projeto.
- `padroes-codigo-projeto.md`: padroes de codigo, arquitetura e UI.
- `padrao-commit.md`: convencao de commits e separacao por responsabilidade.
